# todo_list

这是一个使用vue3编写的用来跟踪待办事项的网页程序
[测试网址](http://todo.windqiu.cn/)

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
